Ce programme est à utiliser à vos risques et périls.
La desinstallation de certaines applications peut rendre votre appareil inutilisable.
Je vous conseille de faire une sauvegarde de votre appareil avant de lancer ce programme.

# Installation

1) Mettre le dossier .désinstallation_auto_android (remettre le "." devant et afficher les fichers cachés) et le fichier désinstallation_auto_android.desktop sur le bureau.

2) Ouvrir un terminal et taper la commande suivante :

chmod u+x /home/$USER/Bureau/désinstallation_auto_android.desktop && chmod u+x /home/$USER/Bureau/.désinstallation_auto_android/*.sh

3) C'est prêt, vous pouvez lancer le programme en cliquant sur le fichier désinstallation_auto_android.desktop
